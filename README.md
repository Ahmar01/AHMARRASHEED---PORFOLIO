## ✏️ Customization Guide

### 1. **Add Your Profile Picture**

Open `professional-portfolio.html` and find:



Replace with:

```html
<div class="profile-picture">
   <img src="1000080278.jpg" alt="AHMAR RASHEED ">
</div>
```

**Photo Tips:**
- Use a professional headshot
- Size: 200x200px or larger
- Format: JPG or PNG
- Keep file in same folder as HTML

### 2. **Update Personal Information**

Find and replace:

```html
<h1>Ahmar Rasheed</h1>
<p class="professional-title">software Engineer | Tech Leader</p>
<p class="hero-description">
    Passionate about building scalable solutions...
</p>
```

### 3. **Update Social Links**

Replace placeholder URLs:

```html
<a href="https://www.linkedin.com/feed/" target="_blank">
<a href="https://github.com/" target="_blank">
<a href="https://twitter.com/yourprofile" target="_blank">
<a href="https://yourwebsite.com" target="_blank">
```

### 4. **Add Your Projects**

Find the portfolio section and update:

```html
<div class="portfolio-item">
    <div class="portfolio-item-image">
        <img src="project1.jpg" alt="Project Name">
    </div>
    <div class="portfolio-item-content">
        <h3>porfolio</h3>
        <p>this is porfolio </p>
        <div class="tags">
            <span class="tag">Technology1</span>
            <span class="tag">Technology2</span>
        </div>
    </div>
</div>
```

### 5. **Update Contact Information**

```html
<a href="mailto:your.email@example.com">📧 ahmarasheed1087@gmail.com</a>
<a href="tel:+92-300-1234567">📱 0326-9512757</a>
```

### 6. **Customize Colors (Optional)**

Find the `:root` CSS variables:

```css
:root {
    --navy: #1a2a47;          /* Main color */
    --dark-navy: #0f1824;     /* Dark variant */
    --gold: #d4a373;          /* Accent color */
    --light-gold: #e8c4a0;    /* Light accent */
    --white: #ffffff;         /* Background */
    --gray: #6b7280;          /* Text color */
}
```
